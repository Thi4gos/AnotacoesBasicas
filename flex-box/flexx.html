Das divs filhas

Order para definir a ordem dos elementos usando, por exemplo, order: 1 para definir que certo elemento ficara a direita de outros menores. Como em uma reta numérica

Flex-grow para os elementos ocuparem de forma proporcional aos valores dos elementos vizinhos, utilizando Flex-grow: 1 assim fazendo com que ocupe todo o espaço possível ao lado. Se por exemplo dois elementos forem flex-grow: 1 e 2 o espaço livro sera dividido em 3 e o primeiro ocupara 1/3 desse.

O Flex-shrink funciona de forma inversa ao Flex-grow diminuindo os elementos de acordo com os seus valores.

Flex-basis pode definir

Flex é uma uma junção ou shorthand do flex-grow, Flex-shrink e Flex-basis sendo definido como, por exemplo, flex: 0 1 auto;.

Align-self alinha o elemento específico com Align-self: flex-end ou start, center;.
